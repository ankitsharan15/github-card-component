A Github card component which takes input from the user and shows the Github user details in a card.
We can add any number of cards on the page.
We can delete a card.
User can sort all the cards by name, location & followers.
Clicking on the card profile pic takes the user to public Github profile page.

Angular js 2.4.0 used .

Steps needed to follow to get the project :
1.install node js
2.npm packgage install
3.clone angular js project from my profile https://github.com/ankitsharan15
